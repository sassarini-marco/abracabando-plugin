## Installazione

**Da Claude Code / claude.ai:**

1. Clicca **Personalizza** → **Sfoglia plugin** → **Personali** → **+** → **Aggiungi marketplace da GitHub**
2. Inserisci `sassarini-marco/abracabando-plugin` e clicca **Sincronizza**
3. Clicca **Installa** sulla scheda AbracaBando

**Da terminale:**

```bash
/plugin marketplace add sassarini-marco/abracabando-plugin
/plugin install abracabando@abracabando
```

> In modalità free installa prima il server dati locale (vedi guida completa). In modalità pro il server è raggiungibile via HTTP — nessuna configurazione aggiuntiva.
